# Evil-Eye Termux Banner by STRIX CYBER COMMUNITY

Installation Step :

1) apt update
2) apt upgrade -y
3) apt install git -y
4) git clone https://github.com/Strix2109/evil-eye
5) cd evil-eye
6) chmod +x requirement.sh
7) chmod +x banner.sh
8) bash requirement.sh
9) bash banner.sh


Uninstallation Step : (Remove Logo From Termux)

1) bash remove.sh

2) Done !!!
